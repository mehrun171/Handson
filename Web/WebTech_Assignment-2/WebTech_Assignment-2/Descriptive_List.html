------- About Technology Tools -------

.NET
A robust framework by Microsoft with the use of C_Sharp (C#) used to build enterprise-grade applications across web, mobile, and desktop.
React
A JavaScript library Framework for building dynamic user interfaces used popularly with MONGODB as MERN Stack.
Machine Learning (ML)
ML stands for Machine Learning, and is the study that uses statistical methods enabling machines to improve with experience.
Deep Learning (DL)
DL stands for Deep Learning, and is the study that makes use of Neural Networks(similar to neurons present in human brain) to imitate functionality just like a human brain.
Angular
A TypeScript-based web framework maintained by Google for building rich, single-page apps, known as MEAN Stack.
Programming Languages
Languages like C, C++, Python, Java, C# etc., that serve as the backbone for software development using coding logic.
